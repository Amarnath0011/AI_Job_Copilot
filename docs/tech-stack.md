# AI Job Copilot - Technology Guide

## FastAPI
High-performance Python framework for REST APIs with automatic validation and OpenAPI documentation.

## LangChain
Framework for orchestrating LLM applications using prompts, retrievers and output parsers.

## ChatPromptTemplate
Builds chat-based prompts using system and human messages.

## PydanticOutputParser
Parses LLM responses into validated Pydantic models.

## Retrieval-Augmented Generation (RAG)
Combines retrieved knowledge with an LLM to produce grounded responses.

## Embeddings
Dense vector representations of text used for semantic search.

## HuggingFaceEmbeddings
Creates embeddings using sentence-transformers/all-MiniLM-L6-v2.

## ChromaDB
Vector database storing embeddings for semantic retrieval.

## Retriever
Fetches the most relevant chunks from ChromaDB.

## Groq
Ultra-low latency inference provider for LLMs.

## LLMService
Application abstraction over AI providers allowing provider switching with minimal code changes.

## RecursiveCharacterTextSplitter
Splits documents into overlapping chunks while preserving context.
